# [Academics] Vision-Servo Underwater Gripping Robot


## 1 Duration
06/00/2021 - 12/00/2022

## 2 Overview
The Underwater Grabbing Robot is designed for 2021 National Innovation Training Project for University. I was responsible for the circuit system design, control algorithm design, and visual servoing algorithm design of the robot. Unlike traditional frame-based underwater grabbing robots, this robot features a four-axis olive-shaped structure that effectively reduces fluid resistance, giving it high maneuverability. The bottom gripper is capable of visual servoing to grasp target objects. The robot's motion control algorithm utilizes a cascaded dual-loop PID controller. The object detection algorithm employs YOLOv5, with the dataset sourced from the OpenImages open-source dataset.
