# [Academics] Vision-Servo Agricultural Quadcopter


## 1 Duration
07/00/2021 - 11/00/2022

## 2 Overview
The Vision Servo Agricultural Quadcopter is the project I designed for National Undergraduate Electronics Design Contest. It ultilizes TM4C123G as the main controller. By using inertial sensors, laser ranging sensors, and optical flow sensors for perception, the UAV enables closed-loop control. Additionally, I employed a camera for visual servo. The control algorithm is based on a dual-loop PID controller, allowing for stable flight and good maneuverability. It is capable of functions such as color block tracking, line following, precision landing, and laser shooting.
